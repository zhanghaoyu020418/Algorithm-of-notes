# 位运算

## 位运算常用模板
```cpp
//统计n的二进制第k位是几
n >> k & 1
//lowbit(x)：每次返回x的最后一位1的数
lowbit(101000) == (1000)2 == 8
lowbit(x) = x & (-x) = x & (~x + 1)
```

### 二进制中1的个数
[二进制中1的个数](https://www.acwing.com/problem/content/803/)

#### lowbit做法

```cpp
#include <iostream>
#include <algorithm>

using namespace std;

int n;

int main ()
{
    cin >> n;
    
    while (n --)
    {
        int x, res = 0;
        cin >> x;
        while (x)
        {
            x -= x & (-x);//lowbit操作
            res ++;//减了几次，res ++几次
        }
        cout << res << ' ';
    }
    return 0;
}
```
#### & >> 做法

while (x)
